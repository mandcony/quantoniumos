Embedding Guide

How to Embed Quantonium OS in Squarespace

Follow these simple steps to embed the Quantonium OS frontend in your Squarespace website:

  1. Open your Squarespace editor
  2. Add an HTML Code Block to your page
  3. Copy and paste the following code:
<iframe src="https://${window.location.host}/frontend" width="100%" height="650px" frameborder="0" scrolling="auto"> </iframe>

The iframe above will embed the full Quantonium OS interface shown below with all functionality.
