Quantonium OS Widget Embed Instructions

This page provides instructions on how to embed the Quantonium OS Cloud Runtime widget into your Squarespace website.

Option 1: Using an Iframe (Easiest)

Insert the following HTML code into a "Code" block in your Squarespace editor:

<iframe src="https://{YOUR-REPLIT-URL}.replit.app/embed" 
       width="100%" 
       height="650" 
       frameborder="0" 
       scrolling="no" 
       allowtransparency="true">
</iframe>
Note: Replace {YOUR-REPLIT-URL} with your actual Replit app URL.

Option 2: Direct HTML/JS Embed (Advanced)

For more control, you can copy the entire HTML/JS content from the widget page and paste it into a "Code" block in Squarespace.

When using this method, you'll need to:

  1. Update the API_URL variable in the JavaScript to point to your deployed Replit app
  2. Ensure the API_KEY matches the one configured on your server
// In your embedded JavaScript
const API_URL = 'https://{YOUR-REPLIT-URL}.replit.app';
const API_KEY = 'default_dev_key'; // Change this to match your server

// Important: API endpoints now have /api prefix
// Example: apiCall('/encrypt', data) will call https://{YOUR-REPLIT-URL}.replit.app/api/encrypt

Security Considerations

When embedding the widget in production:

  1. Set a strong, unique API key on your server using the QUANTONIUM_API_KEY environment variable
  2. Update the API_KEY in the widget code to match
  3. Consider restricting CORS to only your Squarespace domain for added security
Important: Remember that the API key in your JavaScript will be visible to visitors. For high-security applications, consider implementing a more secure authentication method.

Testing Your Embed

To test the widget before embedding it in Squarespace:

  1. Visit the widget page directly
  2. Try each tab's functionality to ensure everything works
  3. Check the browser's developer console for any errors

If you have any questions or encounter issues, please contact support.
