## Key Implementation Details Shown

### Matrix Operations (actual_matrix_multiply.gif)
```
Y[k] = Σ Ψ[k,n] · x[n]  for n=0..N-1
```
Watch each element of the 16×16 matrix multiply with the input vector.

### Phase Computation (phase_computation.gif)
```python
beta = (k / PHI) % 1.0        # Fractional part
theta = 2*pi * beta           # Phase angle
D_phi[k] = exp(1j * theta)    # Golden ratio modulation
chirp[k] = exp(1j * pi * k**2 / N)  # Chirp modulation
```

### Unitarity Test (live_unitarity_test.gif)
```python
X = rft_forward(x)           # Transform
x_recon = rft_inverse(X)     # Inverse
error = ||x_recon - x|| / ||x||  # Relative error
```
All 60 tests achieve ~10⁻¹⁶ error (machine epsilon).

## Mathematical Background

The RFT uses:
- **Golden ratio (φ = 1.618...)** for phase modulation
- **Chirp modulation (σ)** for frequency spreading
- **Orthonormal basis** ensuring Ψ†Ψ = I
- **O(N log N) complexity** via FFT backend

These GIFs demonstrate that the mathematical theory translates into working code with perfect numerical properties.
