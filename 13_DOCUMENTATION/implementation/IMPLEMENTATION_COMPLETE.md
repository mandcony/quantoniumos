## 🎯 **HOW TO USE IN ANY APP:**

### **Method 1: Quick Export (One-liner)**
```python
result = self.os_backend.quick_export('my_app', my_data, show_ui=True)
```

### **Method 2: Direct Export Function**

